MERA Pill Dispenser

The MERA Pill Dispenser is an Internet of Things (IoT) medication organizing system that manages a given user's medications and dispenses the proper dosage based on user input.

Project Concept

There are two main components of the MERA Pill Dispenser system: the physical pill dispenser component and the web interface component.

MERA Pill Dispenser
The physical component of the pill dispenser is an embedded system controlled by a Raspberry Pi that dispenses pills based on the medication schedule and dosage provided by the user.
MERA Website
The web interface component handles communication between the user and the physical unit by allowing users to input their medication schedule and dosage via an interactive website. The information is collected by the website and sent to the physical pill dispenser using HTTP requests so that these instructions can be processed and carried out at real time.

MERA Pill Dispenser Design

There are currently many pill dispensers in the market but these typically require loading predetermined doses into containers to dispense at a later time.

The MERA Pill Dispenser uses a different design where each medication is loaded into a container and pills are counted at the time of dispensing using a sensor.

This system allows greater automatization as users can take individual medications at different times of the day.

Furthermore, medication instructions can easily be updated from anywhere via the MERA website as soon as the doctor increases/decreases a medication dosage or asks the patient to stop taking a medication.
